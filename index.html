Curriculum

What You Will Learn

Hands-on modules across 3 days — from building the chassis to programming autonomous obstacle avoidance.

01

Robot Car Assembly

  • Build a 4-wheel chassis from scratch
  • Mount motors, wheels and battery pack
  • Understand mechanical drive systems
02

Circuits & Electronics

  • Understand voltage, current & connections
  • Wire motor driver module (L298N)
  • Power management and safety basics
03

Arduino Programming

  • Write real C++ code in Arduino IDE
  • Control motors with code logic
  • Debug and upload programs live
04

Sensors & Automation

  • Configure the ultrasonic HC-SR04 sensor
  • Read distance data in real time
  • Program obstacle detection + avoidance logic
